# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Giving and accepting feedback gracefully
- Focusing on what is best for users and the community

Examples of unacceptable behavior:

- Harassment, discrimination, or hateful language
- Personal attacks or trolling
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and contributions that are not aligned with this Code of Conduct.

## Reporting

Please report issues to the maintainers via GitHub issues (use a private channel if sensitive).
